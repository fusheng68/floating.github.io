# JS算法

# 冒泡排序

  • 冒泡排序是一种简单的排序算法，它也是一种稳定排序算法。其实现原理是重复扫描待排序序列，并比较每一对相邻的元素，当该对元素顺序不正确时进行交换。一直重复这个过程，直到没有任何两个相邻元素可以交换，就表明完成了排序。

# js代码

  • 先放上代码，下面会分析算法过程
//从小到大的冒泡排序
for (var i = 0; i < arr.length - 1; i++) {
    for (var j = 0; j < arr.length - 1 - i; j++) {
      // 白话解释：如果前面的数大，放到后面(当然是从小到大的冒泡排序)
      if (arr[j] > arr[j + 1]) {
        let rep = arr[j]
        arr[j] = arr[j + 1]
        arr[j + 1] = rep
      }
    }
  }

# 代码优化

  • 在逻辑分析部分已经实现了冒泡排序。但是代码还有很多可优化的地方

# 循环优化

优化思路

  1. 在循环排序的时候，如果要把100个数字排序，只需要把前面99个数字找到自己的位置，不需要循环全部的数字
  2. 在内层循环中，由于外层循环已经排列了部分数字并放在了末尾，所以可以省略对他们循环
//在循环的时候减去末尾不用排序的
for (var i = 0;i<arr.length -1 ;i++){ 
                              //省略已经排序的数字
  for (var j = 0; j < arr.length -1 - i; j++) {
    if (arr[j] > arr[j + 1]) {
      let rep = arr[j]
      arr[j] = arr[j + 1]
      arr[j + 1] = rep
    }
  }
}

# 代码优化

优化思路

  1. 使用es6的新语法省略代码
//在循环的时候减去末尾不用排序的
for (var i = 0;i<arr.length -1 ;i++){ 
  //省略已经排序的数字
  for (var j = 0; j < arr.length -1 - i; j++) {
    if (arr[j] > arr[j + 1]) {
      // 数组的解构赋值
      [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
    }
  }
}

# 动画效果

  • 由于我是用vue写的，只有在代码执行完才更新页面，导致看不到排序动画的效果，所以在上面代码的基础上加上延时，让排序的过程可以展示出来
  • 这里使用asyns,await,和Promise配合使用
function wait(i) {
  return new Promise((res) => {
    setTimeout(() => {
      console.log(i);
      res();
    });
  });
}
async function ranking() {
  for (var i = 0; i < arr.length - 1; i++) {
    for (var j = 0; j < arr.length - 1 - i; j++) {
      // 白话解释：如果前面的数大，放到后面(当然是从小到大的冒泡排序)
      if (arr[j] > arr[j + 1]) {
        await wait(i);
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
      }
    }
  }
}

# 完整vue代码(vue3)

<script setup>
import { reactive } from "vue";
let arr = reactive([]);
for (let i = 0; i < 100; i++) {
  arr.push(i);
}
arr.sort((e) => Math.random() - 0.5);
function wait(i) {
  return new Promise((res) => {
    setTimeout(() => {
      console.log(i);
      res();
    });
  });
}
async function ranking() {
  for (var i = 0; i < arr.length - 1; i++) {
    for (var j = 0; j < arr.length - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        await wait(i);
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
        
      }
    }
  }
}
</script>

<template>
  <div class="list">
    <button class="star" @click="ranking">开始</button>
    <div
      class="con"
      v-for="item in arr"
      :key="item"
      :style="{ height: item+1 + 'vh' }"
    ></div>
  </div>
</template>

<style scoped>
.list {
  width: 100%;
  height: 100vh;
  display: flex;
  align-items: flex-end;
  position: relative;
}
.star {

  top: 20px;

}
.con {
  width: 1%;
  background-color: antiquewhite;
  box-sizing: border-box;
  border: 1px solid #777;
  transition: transform 0.5s;
}
</style>
